WordWin 设置

API 配置

部分模型默认会开启 reasoning/thinking。WordWin 默认关闭；需要更强推理时再打开。

当前仅内置支持 OpenAI、阿里云百炼和火山方舟。

并发越高，速度越快，但更容易触发限流或消耗额度。建议 1-10。

翻译范围

全面翻译模式会处理导航、按钮、目录、页脚和正文，并尽量给短文本也加中文提示；正文翻译模式主要处理文章正文。

阅读等级

L1 标注最多，L5 标注最少。

系统 Prompt

这是发送给 LLM 的指令。你可以编辑它来自定义标注行为（比如加上词性、改用英文释义等）。

查看 L1-L5 的完整等级说明

L1

能力参考：约等于 IELTS 2 分及以下

阅读特点：几乎没有英语阅读基础，只认识最基本的功能词和代词。

标注策略：绝大多数实义词都需要标注，每段 10-20 个词。

L2

能力参考：约等于 IELTS 4 分

阅读特点：认识常见简单词汇，但中等难度词和短语容易卡住。

标注策略：标注中低频词、抽象词、较长词和不常见短语，每段 6-12 个词。

L3

能力参考：约等于 IELTS 5.5 分

阅读特点：能读懂大部分日常内容，但学术词汇和书面表达会卡住。

标注策略：只标注学术词、专业表达和语义不直观的短语，每段 3-6 个词。

L4

能力参考：约等于 IELTS 7 分

阅读特点：能流畅阅读大部分英文内容，词汇量较大。

标注策略：只标注低频专业术语和罕见书面表达，每段最多 1-2 个词。

L5

能力参考：约等于 IELTS 8 分及以上

阅读特点：接近母语阅读水平，几乎所有英文内容都能直接理解。

标注策略：只标注极罕见的专业术语，绝大多数段落无需标注。

Token 用量统计

统计来自各服务商返回的 usage 字段。少数接口不返回 usage 时，只记录请求次数。

清空后会删除本机保存的 token 用量记录，此操作不可恢复。
